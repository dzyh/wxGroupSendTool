# wxGroupSendTool
##### 基于wxpy一个简单的群发工具

安装python打包插件[PyInstaller](https://github.com/pyinstaller/pyinstaller/blob/0f31b35fe9/doc/installation.rst)
```python
pip install PyInstaller
```
打包
```python
python -F groupSendTool.py 
```
